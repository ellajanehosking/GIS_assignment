# Sandprawn distribution and ecosystem protection level

This assignment is the for an honours module on GIS in R. For this assignment I created an interactive map of *Kraussillichirus kraussi* observations in South Africa colour coded by ecosystem protection level. Each point on the map has a clickable URL to iNaturalist for more information on each observation.

#### Ecosystem protection levels:
* Not protected
* Hardly protected
* Poorly protected
* Moderately protected
* Well protected

#### Data sources:
* iNaturalist observations
* South African National Biodiversity Institute - coastal ecosystem levels of protection
